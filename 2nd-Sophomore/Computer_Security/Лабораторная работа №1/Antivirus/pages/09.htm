Глоссарий

 

ICSA

Международная Ассоциация по Компьютерной Безопасности.

 

CERT

Координационный центр группы реагирования на экстренные ситуации, связанные с компьютерами.

 

Саморепликация

Создание себе подобной структуры.

 

BIOS

(Basic

Input/Output System)

 

Базовая система ввода вывода – используется различными программами, для работы с аппаратными ресурсами компьютера (в том числе: жестким диском, дисководом, принтером, монитором и т.д.) без участия операционной системы.

 

MBR

(Master Boot Record)

 

Главная загрузочная запись – содержит информацию о разделах жесткого диска, а также адрес первого сектора того раздела, который является активным (там расположен Boot record).

 

BR

(Boot record)

 

Загрузочная запись – содержит информацию о месторасположении и последовательности загрузки файлов операционной системы.

 

Partition Table

Таблица разделов жесткого диска – содержит информацию об имеющихся на физтческом диске логических разделах (дисках), а также об их расположении на физическом диске.

 

FAT

(File Allocation Table)

 

Таблица размещения файлов – содержит информацию о размещении частей каждого файла на жестком диске, а также устанавливает их последовательность, т.к. на диске файлы чаще всего хранятся

 не как единое целое, а как цепочка логически связанных между собой кусков, раскиданных по всему диску.

 

«Псевдосбойные» кластеры

Место на жестком диске помеченное как непригодное для хранения информации, но фактически являющееся нормальным. Обычно создаются вирусами для того чтобы спрятать какие либо данные, либо чтобы уменьшить объем доступного пользователю места для хранения информации.
